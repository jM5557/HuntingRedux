Mod Your Own Textures!

You will first need to EXTRACT the textures from whichever PPSSPP game you are running. Run your game and go to an area that you want to rip textures from.

Then Save State and afterwards navigate to:

Settings Tools Developer Tools Save new textures

Now Load State and let the game save the new textures from the game on-screen.

These textures can be found inside:

memstick PSP TEXTURES YOUR_GAME_ID new

Open up the textures you want to edit in Photoshop or GIMP. Once you've made your edits, save them to:

memstick PSP TEXTURES YOUR_GAME_ID

Then check the option:

Settings Tools Developer Tools Replace Textures



Make sure to uncheck Save new textures

Voila! Your new textures are now loaded.
