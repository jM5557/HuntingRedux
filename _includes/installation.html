Installation

Download PPSSPP and the latest release of HuntingRedux.

Use your preferred zip extractor to extract the TEXTURES folder into the following location of your PPSSPP installation:

memstick/PSP/TEXTURES/ULUS10391

On MacOS, you can locate this folder in:

~/.config/ppsspp/PSP/TEXTURES/ULUS10391

In PPSSPP, check off the following option:

Settings Tools Developer Tools Replace Textures
