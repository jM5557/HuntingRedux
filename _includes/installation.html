Installation

Download and install PPSSPP along with the latest {{ include.gameSubname }} release.

Use a zip extractor to extract the contents of the project into a separate folder. We recommend 7zip.

Extract the file contents into the following location of your PPSSPP installation folder:

memstick/PSP/TEXTURES/ULUS10391

On MacOS, you can locate this folder in:

~/.config/ppsspp/PSP/TEXTURES/ULUS10391

Make sure to check this option in PPSSPP:

Settings Tools Developer Tools Replace Textures
